サウンドマネージャーの使い方を説明させて頂きます。
I will explain how to use the Sound Manager.

①BGMがついているObjectを、TargetBGMSoundObjectに設定します。(GameObject)
  An Object that BGM is on, set to TargetBGMSoundObject. (GameObject)

②BGMを設定するSliderをTargetBGMSliderに設定します。(Slider)
  You set the Slider to set the BGM to TargetBGMSlider. (Slider)

③SoundEffectがついているObjectを、TargetEffectSoundObjectに設定します。(GameObject)
  An Object that SoundEffec is on, set to TargetEffectSoundObject. (GameObject)

④SoundEffecを設定するSliderをTargetEffectSliderに設定します。(Slider)
  You set the Slider to set the SoundEffec to TargetEffectSlider. (Slider)

⑤設定完了です。
  Setup is complete.
